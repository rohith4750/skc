# Public Directory

This directory contains static assets for the website.

## Logo Image

Please place your SKC CATERERS logo image file in this directory with one of these names:
- `logo.png` (recommended)
- `logo.jpg`
- `logo.svg`

The image should be:
- **For website logo**: At least 200x200 pixels (square format works best)
- **For favicon**: At least 32x32 pixels, but 256x256 is recommended

If you have separate files for different sizes, you can name them:
- `logo.png` - Main logo (200x200 or larger)
- `favicon.png` - Tab icon (32x32 or 256x256)

The system will automatically use these images once they are placed in this directory.
